FreeCodeCamp API Basejump: URL Shortener Microservice

User stories:

  • I can pass a URL(without https or http) as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

localhost:3000/new/www.google.com

Example creation output

{ "original_url":"www.google.com", "short_url":"localhost:3000/1234" }
